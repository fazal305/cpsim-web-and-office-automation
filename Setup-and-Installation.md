# Setup and Installation

## Requirements

- A modern web browser such as Chrome, Edge, Firefox, or Safari
- VS Code or another text editor for editing web files
- Microsoft Office or a compatible office suite for `.xlsx`, `.docx`, and `.pptx` files

## Run Web Projects

1. Download or clone the repository.
2. Open the repository folder.
3. Go to `web-projects/`.
4. Double-click any `.html` file to open it in a browser.

Recommended files to preview first:

- `web-projects/quiz-app.html`
- `web-projects/survey-form.html`
- `web-projects/restaurant-menu.html`
- `web-projects/bill.html`
- `web-projects/todolist.html`

## Open Office Automation Files

1. Go to `office-automation/`.
2. Open any Excel, Word, or PowerPoint file.
3. Review the formulas, layout, and document structure.

## Notes

This repository does not require a package manager, backend server, or installation script.
