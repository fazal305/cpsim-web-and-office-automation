# Contributing

Thank you for checking out this repository.

This is a beginner-friendly learning archive, so contributions should keep the examples simple, readable, and easy to run.

## Guidelines

- Keep HTML, CSS, and JavaScript beginner-friendly.
- Avoid adding unnecessary frameworks or build tools.
- Use clear file names.
- Add comments only when they explain useful learning points.
- Keep Office files organized inside `office-automation/`.
- Keep web examples organized inside `web-projects/`.
- Update the README when adding a new major example.
